## 内容简介
{% note %}
陕籍在京青年评论家李建军，日前于京城某报大篇幅“炮轰”作家贾平凹的长篇新作《秦腔》，称其为“一部粗俗的失败之作”，引起业界的关注，20日，“炮轰”对象贾平凹接受了本报记者的独家专访，一改往常沉默之状，有问必答，高调回应。
{% endnote %}

## 作者简介
{% note %}
马尼尼为，马来西亚华人，本名林婉文。著有绘本、散文、诗集，曾获金鼎奖、台北文学奖、全球华文文学星云奖、OPENBOOK好书奖、钟肇政文学奖等多项大奖。三十岁后重拾创作，有一只叫阿美的爱猫。
{% endnote %}
