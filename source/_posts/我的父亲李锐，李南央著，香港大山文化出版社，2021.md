## 内容简介
{% note %}
《我的父亲李锐》一书与香港大山文化出版社所签五年出版合同期限已过，应友人请求，整理出此电子版供读者传阅，本人不保留版权。为方便打印成册，电子版保留了纸质书的开本及边框留距，仅字体、排版与纸质书略有不同。纸质书出版后，陆续收到的一些朋友的指误和建议，都一一在电子版中作了勘误和增补。特此致谢！

李南央
2021年9月
于美国旧金山湾区家中
{% endnote %}

## 作者简介
{% note %}
2014年10月7日，五柳村网刊出奚青、赵来群和黄以群三人化名文章《李锐访谈录——关于<我的父亲李锐>一书》，它成为张玉珍在中国起诉及在美国反诉李南央同斯坦福大学的重要证据。

父亲在“访谈录”中说：“我们这些老家伙是救党派，李南央对共产党是整个否定的。”但是父亲让我他一生的个人文字代为捐赠给斯坦福大学胡佛研究所，成为社会公器。父女史观相通，故而相谋。
{% endnote %}
